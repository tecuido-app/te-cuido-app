# Smart Contract — TE CUIDO

Anchor program that stores every agent action as an immutable on-chain audit log.

## Deploy on Solana Playground (no local install required)

1. Open **https://beta.solpg.io**
2. Create a new project: **"Create"** button → **Anchor**
3. Paste the contents of `lib.rs` into the editor (replace the default example)
4. Click **"Build"** bottom left — should compile in ~30 seconds
5. Click **"Deploy"** — Playground automatically gives you free SOL
6. Copy the **Program ID** shown in the console (e.g. `7xKp9mZqA...`)
7. Paste the Program ID in:
   - Line 11 of `lib.rs`: change `declare_id!("11111111111111111111111111111111")` to your real ID
   - `.env` at the project root: `PROGRAM_ID=...` and `NEXT_PUBLIC_PROGRAM_ID=...`
8. **Build** + **Deploy** again (required after changing `declare_id!`)
9. Download the IDL: **"IDL"** tab → **"Download"** → save as `solana/idl.json`

Done. The contract is live on devnet.

## Verify it works

In Playground, **"Test"** or **"Anchor"** tab → **Instructions**:
- Try `register_event` with `event_id=12345`, `event_type=0` (Fall), `severity=3`, `value=600`
- Should return a successful transaction
- Check the created account in the Explorer

## Program instructions

| Instruction | Args | Signer |
|---|---|---|
| `register_event` | `event_id (u64)`, `event_type (u8)`, `severity (u8)`, `value (i64)` | agent |
| `register_action` | `action_type (u8)`, `contact_index (u8)` | agent |
| `confirm_wellbeing` | (no args) | agent (in MVP — in prod, the family member) |

## Numeric codes (no enums for client simplicity)

**EventType:**
- `0` = Fall
- `1` = LowHR (bradycardia)
- `2` = LowSpO2 (hypoxia)

**ActionType:**
- `0` = GracePeriod (60s window)
- `1` = AIDismissed (dismissed by AI as false positive)
- `2` = NotifiedContact (Telegram sent)
- `3` = Escalated (escalated to emergency)
- `4` = WellbeingConfirmed (contact confirmed OK)
- `5` = Resolved (closed)

## EventLog PDA

```
seeds = [b"event", event_id_le_bytes(8)]
```

`event_id` is generated by the client (e.g. `int(time.time() * 1000)`). Guarantees uniqueness by timestamp.
